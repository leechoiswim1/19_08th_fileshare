## 6. 💿 설치 및 실행 방법

Project Clone

```bash
$ git clone https://github.com/wantedPreOnboarding/19_08th_fileshare.git

```

Project Setup

```bash
$ yarn install
```

Project Start For Development

```bash
$ yarn start
```
